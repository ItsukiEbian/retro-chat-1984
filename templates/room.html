スマホ・他端末でカメラが使えない場合

ブラウザのセキュリティのため、HTTP（例: 192.168.x.x）ではカメラが使えません。次のいずれかで対応できます。

  • ngrok / cloudflared で HTTPS のURLを発行して、そのURLでアクセスする（推奨）。手順はプロジェクトの README を参照。
  • Chrome（Android等）: chrome://flags で Insecure origin treated as secure を検索し、http://あなたのPCのIP:10000 を追加して有効化し、ブラウザを再起動。
